Welcome to Katik

'Katik' adalah kalkulator web interaktif yang didesain khusus untuk membantu Anda menghitung luas dan keliling segitiga. Pengguna hanya perlu memasukkan panjang sisi, alas, dan tinggi yang relevan. Kalkulator akan langsung menampilkan hasilnya. Proyek ini akan berfokus pada penyediaan antarmuka yang bersih dan mudah digunakan, memastikan bahwa perhitungan dapat dilakukan dengan cepat dan akurat. Dibangun menggunakan HTML, CSS, dan JavaScript, 'Katik' akan menjadi alat praktis yang membutuhkan perhitungan segitiga secara instan.

Hitung Luas Segitiga

Rumus Luas Segitiga :

Luas = 1/2 x a x t

Dimana :

  • L = Luas
  • a = Alas
  • t = Tinggi

Hitung Keliling Segitiga

Rumus Keliling Segitiga :

Keliling = a + b + c

Dimana :

  • K = Keliling
  • a = Sisi 1
  • b = Sisi 2
  • c = Sisi 3
